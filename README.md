# Student Performance Analysis

## Overview
This project analyzes the factors that affect student exam performance 
using a dataset of 1000 students. The analysis explores how gender, 
test preparation, parental education, and socioeconomic status relate 
to academic success.

## Key Questions
- Does gender affect exam scores?
- Does completing a test preparation course improve performance?
- How does parental education level relate to student scores?
- Does socioeconomic status (lunch program) impact performance?

## Key Findings
- Female students scored approximately 4 points higher on average
- Students who completed test preparation scored significantly better
- Socioeconomic status was the strongest predictor — 10 point difference
- Higher parental education generally leads to better student scores
